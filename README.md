# Обработка Word-файлов

Приложение "Обработка Word-файлов" помогает извлекать имена персонажей из таблиц Word-документов, подсчитывать частоту упоминаний и формировать аккуратный отчёт.

## Функционал

* Поддержка файлов формата .docx и обработка всех таблиц внутри документа
* Настраиваемый номер столбца, откуда считываются имена (по умолчанию второй)
* Формирование структурированной сводной таблицы «Персонаж / Количество» с выделением топа
* Опция пропуска строк-заголовков при чтении таблиц
* Копирование всех исходных таблиц в результирующий документ
* Обработка одиночных файлов и пакетная обработка множества документов
* Свободный выбор имени и места сохранения отчётных файлов

## Как использовать

1. Запустите `window-strat-app.py` (обычно достаточно `python window-strat-app.py`).
2. В поле «Номер столбца» укажите, из какого столбца таблиц нужно брать имена (1 = первый столбец).
3. Отметьте «Пропускать первую строку таблиц», если в ваших таблицах первая строка содержит заголовки и не должна участвовать в подсчёте.
4. Выберите один из режимов работы:
   * **«Выбрать один файл»** — выберите документ и затем укажите путь сохранения отчёта.
   * **«Обработать несколько файлов»** — выберите группу документов и папку, куда будут сохранены отчёты.
5. Дождитесь сообщения об успешном завершении. При пакетной обработке программа покажет статус и список всех сохранённых файлов.

## Советы и подсказки

* Если не уверены в структуре таблицы, откройте документ в Word и посчитайте колонки вручную, после чего введите соответствующий номер в интерфейсе (значение 1 означает первый столбец).
* При установленном флажке «Пропускать первую строку» не учитываются строки-заголовки, поэтому используйте его для таблиц, где надписи «Имя», «Персонаж» и т. п. вынесены в отдельную строку.
* В пакетном режиме удобнее сначала выбрать все файлы в одном каталоге, а затем указать отдельную папку назначения — программа создаст одноимённые отчёты с суффиксом `_result` и предупредит о возможном перезаписывании.
* Если нужно выделить только лидеров, можно отредактировать итоговую таблицу в Word: например, оставить верхние N строк или построить диаграмму на основе подсчитанных значений.

## Требования

* Python 3.10+
* Библиотека [python-docx](https://python-docx.readthedocs.io/)
* Tkinter (входит в стандартную поставку Python для Windows и Linux; для macOS может потребоваться установка `python-tk`)
