# Обработка Word-файлов

Приложение "Обработка Word-файлов" позволяет автоматически обрабатывать Word-документы и выделять персонажей из второго столбца таблиц.

## Функционал

* Поддержка файлов формата .docx
* Выделение персонажей из второго столбца таблиц
* Сохранение результата в виде нового документа

## Как использовать

1. Загрузите Word-документ (.docx)
2. Нажмите на кнопку "Выбрать файл и обработать"
3. Приложение автоматически обрабатывает документ и выделяет персонажей
4. Результат сохраняется в виде нового документа

## Требования

* Python 3.x
* Библиотека docx для работы с Word-документами
